# knodeip

```txt
$ knodeip --help
Query the external ip addresses of a pod's node

Usage:
  knodeip [flags]

Flags:
  -h, --help                help for knodeip
      --kubeconfig string   Path to the kubeconfig file. If empty will use the in-cluster mode.
      --namespace string    Cluster namespace.
      --pod string          Pod name to query.
```
